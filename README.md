# Notes App

A fullstack note taking application that uses Node and Express.

## Features

- Add new notes
- Edit existing notes
- Delete notes
- Notes are saved to a JSON file so that the user can update the notes at a later date. 

## How to run locally

1. Install dependencies:
```
npm install
```

2. Start the server:
```
npm start
```

3. Open your browser and go to: http://localhost:3001
